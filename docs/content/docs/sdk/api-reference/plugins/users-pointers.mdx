## Overview

The [WeaveUsersPointersPlugin](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/plugins/users-pointers/users-pointers.ts)
class that enables real-time cursor sharing across all connected users on the collaborative canvas. It displays live
pointer positions, often tagged with user metadata like name or color, allowing participants to see each otherâ€™s
activity and collaborate more intuitively.

This plugin helps create a shared sense of presence, making collaborative tools feel dynamic, interactive, and
truly multi-user.

The class extends the [WeavePlugin](/docs/sdk/api-reference/plugins/plugin) class

## Name

This plugin `name` property value is `usersPointers`.
