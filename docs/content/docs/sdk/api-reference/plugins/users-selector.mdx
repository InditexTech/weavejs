## Overview

The [WeaveUsersSelectionPlugin](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/plugins/users-selector/users-selector.ts)
class that enables real-time user selection sharing across all connected users on the collaborative canvas. It displays live
what each user is selecting, allowing participants to see each otherâ€™s activity and collaborate more intuitively.

This plugin helps create a shared sense of presence, making collaborative tools feel dynamic, interactive, and
truly multi-user.

The class extends the [WeavePlugin](/docs/sdk/api-reference/plugins/plugin) class
