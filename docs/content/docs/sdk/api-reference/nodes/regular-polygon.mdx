## Overview

The [WeaveRegularPolygonNode](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/nodes/regular-polygon/regular-polygon.ts)
class represents a basic regular polygon shape within the Weave.js canvas.

Built on top of Konvaâ€™s [RegularPolygon](https://konvajs.org/api/Konva.RegularPolygon.html) class shape under the hood, the WeaveRegularPolygonNode
offers a simple yet powerful primitive that supports full interaction, styling, and real-time synchronization.

The class extends the [WeaveNode](/docs/sdk/api-reference/nodes/node) class
