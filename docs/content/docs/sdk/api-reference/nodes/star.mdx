## Overview

The [WeaveStarNode](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/nodes/star/star.ts)
class represents a basic star shape within the Weave.js canvas.

Built on top of Konvaâ€™s [Star](https://konvajs.org/api/Konva.Star.html) class shape under the hood, the WeaveStarNode
offers a simple yet powerful primitive that supports full interaction, styling, and real-time synchronization.

The class extends the [WeaveNode](/docs/sdk/api-reference/nodes/node) class

## Type

This node `nodeType` property value is `star`.

```shell
import { WeaveStarNode } from "@inditextech/weave-sdk";
```
