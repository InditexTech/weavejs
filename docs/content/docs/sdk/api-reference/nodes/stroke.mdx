## Overview

The [WeaveStrokeNode](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/nodes/stroke/stroke.ts)
class represents a multi-point continuous line or path on the Weave.js canvas. It is a versatile and essential node type for
building tools like freehand drawing.

Built on top of Konva’s [Shape](https://konvajs.org/api/Konva.Shape.html) class shape under the hood, the WeaveStrokeNode
offers full control over the line’s geometry, style, and behavior—all synchronized in real time across users.

The class extends the [WeaveNode](/docs/sdk/api-reference/nodes/node) class

## Type

This node `nodeType` property value is `stroke`.

## Import

```shell
import { WeaveStrokeNode } from "@inditextech/weave-sdk";
```

## Instantiation

```ts
new WeaveStrokeNode(params?: WeaveStrokeNodeParams);
```
