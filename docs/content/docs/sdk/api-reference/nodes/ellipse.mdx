## Overview

The [WeaveEllipseNode](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/nodes/ellipse/ellipse.ts)
class represents a basic ellipse shape within the Weave.js canvas.

Built on top of Konva’s [Ellipse](https://konvajs.org/api/Konva.Ellipse.html) class shape under the hood, the WeaveEllipseNode
offers a simple yet powerful primitive that supports full interaction, styling, and real-time synchronization.

Ellipse nodes support an optional **text label** that is displayed centred inside the shape.
The label can be edited interactively by double-clicking the ellipse while it is selected.
All label appearance properties (font, colour, alignment, padding, …) are configurable via the
`label*` props described below.

The class extends the [WeaveNode](/docs/sdk/api-reference/nodes/node) class

## Type

This node `nodeType` property value is `ellipse`.

## Import

```shell
import { WeaveEllipseNode } from "@inditextech/weave-sdk";
```
