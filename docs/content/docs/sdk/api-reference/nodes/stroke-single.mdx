## Overview

The [WeaveStrokeSingleNode](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/nodes/stroke-single/stroke-single.ts)
class represents a single stroke line on the Weave.js canvas. It is a versatile and essential node type for
building tools like freehand drawing. It supports also define tips for both stroke ends as:

- Arrow
- Circle
- Square

The class extends the [WeaveNode](/docs/sdk/api-reference/nodes/node) class

## Type

This node `nodeType` property value is `stroke-single`.

## Import

```shell
import { WeaveStrokeSingleNode } from "@inditextech/weave-sdk";
```

## Instantiation

```ts
new WeaveStrokeSingleNode(params?: WeaveStrokeSingleNodeParams);
```
