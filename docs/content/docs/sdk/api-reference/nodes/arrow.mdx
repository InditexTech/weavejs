## Overview

The [WeaveArrowNode](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/nodes/arrow/arrow.ts)
class represents a basic arrow shape within the Weave.js canvas.

Built on top of Konvaâ€™s [Arrow](https://konvajs.org/api/Konva.Arrow.html) class shape under the hood, the WeaveArrowNode
offers a simple yet powerful primitive that supports full interaction, styling, and real-time synchronization.

Arrow are ideal for:

- Point to important stuff on whiteboards.
- Connect flows of ideas.

The class extends the [WeaveNode](/docs/sdk/api-reference/nodes/node) class

## Type

This node `nodeType` property value is `arrow`.
