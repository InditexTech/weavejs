## Overview

The [WeaveAlignNodesToolAction](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/actions/align-nodes-tool/align-nodes-tool.ts)
class that allows users to align selected nodes vertically or horizontally.

The class extends the [WeaveAction](/docs/sdk/api-reference/actions/action) class.

## Name

This action `name` property value is `alignNodesTool`.
