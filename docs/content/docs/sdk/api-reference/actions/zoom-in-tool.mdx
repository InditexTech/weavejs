## Overview

The [WeaveZoomInToolAction](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/actions/zoom-in-tool/zoom-in-tool.ts)
class that allows users to incrementally zoom in on the canvas Stage. It provides
a simple, user-friendly way to enlarge the view, helping users focus on fine
details within a complex visual environment.

This action works in coordination with the [WeaveStageZoomPlugin](/docs/sdk/api-reference/plugins/stage-zoom)
plugin and ensures a smooth zooming experience without disrupting the
layout or collaboration session.

The class extends the [WeaveAction](/docs/sdk/api-reference/actions/action) class.

## Name

This action `name` property value is `zoomInTool`.
