## Overview

The [WeaveConnectorToolAction](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/actions/connector-tool/connector-tool.ts)
class allows users to add connector to the canvas by click-and-click onto pre-defined anchors on the canvas elements.

The class extends the [WeaveAction](/docs/sdk/api-reference/actions/action) class.

## Name

This action `name` property value is `connectorTool`.

## Import

```shell
import { WeaveConnectorToolAction } from "@inditextech/weave-sdk";
```

## Instantiation

```ts
new WeaveConnectorToolAction(params?: WeaveConnectorToolActionParams);
```
