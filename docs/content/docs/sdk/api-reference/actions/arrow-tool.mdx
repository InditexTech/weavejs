## Overview

The [WeaveArrowToolAction](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/actions/arrow-tool/arrow-tool.ts)
class that allows users to draw precise, vector-based arrows by clicking to place
points, the pen tool emphasizes structured arrow creation, making it ideal for diagrams, connectors,
wireframes, and flow-based interactions.

Each completed arrow becomes a [WeaveArrowNode](/docs/sdk/api-reference/nodes/arrow),
synced in real time across all collaborators.

The class extends the [WeaveAction](/docs/sdk/api-reference/actions/action) class.

## Name

This action `name` property value is `arrowTool`.

import { WeaveArrowToolAction } from "@inditextech/weave-sdk";
