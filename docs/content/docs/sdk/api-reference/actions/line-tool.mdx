## Overview

The [WeaveLineToolAction](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/actions/line-tool/line-tool.ts)
class that allows users to draw a precise, vector-based line by click-and-drag to define the line.

The line becomes a [WeaveLineNode](/docs/sdk/api-reference/nodes/line),
synced in real time across all collaborators.

The class extends the [WeaveAction](/docs/sdk/api-reference/actions/action) class.

## Name

This action `name` property value is `lineTool`.

## Import

```shell
import { WeaveLineToolAction } from "@inditextech/weave-sdk";
```

new WeaveLineToolAction(params?: WeaveLineToolActionParams);
