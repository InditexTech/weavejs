## Overview

The [WeaveStrokeToolAction](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/actions/stroke-tool/stroke-tool.ts)
class that allows users to draw a precise, vector-based line by click-and-drag to define the line.

The line becomes a [WeaveStrokeSingleNode](/docs/sdk/api-reference/nodes/stroke-single),
synced in real time across all collaborators.

The class extends the [WeaveAction](/docs/sdk/api-reference/actions/action) class.

## Name

This action `name` property value is `strokeTool`.
