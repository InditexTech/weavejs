## Overview

The [WeaveMeasureToolAction](https://github.com/InditexTech/weavejs/blob/main/code/packages/sdk/src/actions/measure-tool/measure-tool.ts)
class that allows users to add measure to the canvas by click-and-click to define the measure points.

The class extends the [WeaveAction](/docs/sdk/api-reference/actions/action) class.

## Name

This action `name` property value is `measureTool`.
