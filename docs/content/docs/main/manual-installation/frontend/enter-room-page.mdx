In this guide, we will walk through the process of building the **Enter Room page**:

![Enter room page](/images/enter-room-page.png)

This page allows users to define the room they wish to join and set their username. 
Let's build it step by step.

## Prerequisites

Before you begin, ensure that you have completed the [frontend installation](/docs/main/manual-installation/frontend/enter-room-page) guide, up to the *"Set up the Enter Room page"* step.

### Install the UI building block components

Set up the necessary [Shadcn/UI](https://ui.shadcn.com/docs) components for the **Enter Room** page.

We will be using the [Shadcn/UI CLI](https://ui.shadcn.com/docs/cli) to install the UI components. Accept
all the defaults from the CLI options.

In the root directory of your project, follow these steps:

- Install the [Form](https://ui.shadcn.com/docs/components/form) component:

  ```shell
  npx shadcn@latest add form
  ```

- Install the [Input](https://ui.shadcn.com/docs/components/input) component:

  ```shell
  npx shadcn@latest add input
  ```

</div>
