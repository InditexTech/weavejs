The Video Node allows you to render and play videos directly onto the canvas. Built on top
of Konvaâ€™s [Image](https://konvajs.org/api/Konva.Image.html) shape, and [Konva Animation](https://konvajs.org/api/Konva.Animation.html)
it supports properties play, pause and progress tracking.

## Usage

<div className="fd-steps">

<div className="[&_h3]:fd-step">

### Import the Node

Start by importing the node:

```ts
import { WeaveVideoNode } from "@inditextech/weave-sdk";
```

</div>

<div className='[&_h3]:fd-step'>

### Register the Node

Then register the node on the [Weave](/docs/sdk/api-reference/weave) class instance.

```ts
const instance = new Weave({
  ...
  nodes: [
    ...,
    new WeaveVideoNode(), // [!code ++]
  ]
})
```

</div>

<div className='[&_h3]:fd-step'>

### Use the node

Once the node is registered you can use it on [Actions](/docs/main/build/actions) or
[Plugins](/docs/main/build/plugins) or even other [Nodes](/docs/main/build/nodes).

We provide an action named [Video Tool](/docs/main/build/actions/video-tool) that allows
users to add Video (one-by-one) nodes to the application.

</div>
