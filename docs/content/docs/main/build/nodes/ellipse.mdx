The Ellipse node is a simple, versatile primitive shape used to render ellipse
elements on the canvas. Built on top of Konvaâ€™s [Ellipse](https://konvajs.org/api/Konva.Ellipse.html),
it supports customizable properties such as position, axis radius, fill color, stroke, radius (on both axis),
and more.

They can also respond to interactions like dragging, resizing, or selection.

## Usage

<div className="fd-steps">

<div className="[&_h3]:fd-step">

### Import the Node

Start by importing the node:

```ts
import { WeaveEllipseNode } from "@inditextech/weave-sdk";
```

</div>

<div className='[&_h3]:fd-step'>

### Register the Node

Then register the node on the [Weave](/docs/sdk/api-reference/weave) class instance.

```ts
const instance = new Weave({
  ...
  nodes: [
    ...,
    new WeaveEllipseNode(), // [!code ++]
  ]
})
```

</div>

<div className='[&_h3]:fd-step'>

### Use the node

Once the node is registered you can use it on [Actions](/docs/main/build/actions) or
[Plugins](/docs/main/build/plugins) or even other [Nodes](/docs/main/build/nodes).

We provide an action named [Ellipse Tool](/docs/main/build/actions/ellipse-tool) that
allows users to add Ellipse nodes to the application.

</div>
