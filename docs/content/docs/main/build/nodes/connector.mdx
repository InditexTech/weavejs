![Connector on the Weave.js showcase](/images/nodes/connector.png)

The Connector node is a complex node used to render a line between two elements anchors, if the elements move
the line also does automatically.

## Usage

<div className="fd-steps">

<div className="[&_h3]:fd-step">

### Import the Node

Start by importing the node:

```ts
import { WeaveConnectorNode } from "@inditextech/weave-sdk";
```

</div>

<div className='[&_h3]:fd-step'>

### Register the Node

Then register the node on the [Weave](/docs/sdk/api-reference/weave) class instance.

```ts
const instance = new Weave({
  ...
  nodes: [
    ...,
    new WeaveConnectorNode(), // [!code ++]
  ]
})
```

</div>

<div className='[&_h3]:fd-step'>

### Use the node

Once the node is registered you can use it on [Actions](/docs/main/build/actions) or
[Plugins](/docs/main/build/plugins) or even other [Nodes](/docs/main/build/nodes).

We provide an action named [Connector Tool](/docs/main/build/actions/connector-tool) that
allows users to add Connector nodes to the application in a simple and intuitive way.

</div>

</div>
