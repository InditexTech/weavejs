## Introduction

This plugin adds smart alignment assistance by enabling nodes to snap to guides or
other nodes as they are moved or resized. It helps users place elements with
precision, improving layout consistency and visual structure.

Itâ€™s especially useful in design tools, diagram editors, and any interface where
visual alignment matters.

## Dependencies

This plugin needs also registered on the Weave instance the following:

- [Nodes selection](/docs/main/build/plugins/nodes-selection) plugin.

## Usage

<div className="fd-steps">

<div className="[&_h3]:fd-step">

### Import the Plugin

Start by importing the plugin:

```ts
import { WeaveNodesSnappingPlugin } from "@inditextech/weave-sdk";
```

</div>

<div className='[&_h3]:fd-step'>

### Register the Plugin

Then register the plugin on the [Weave](/docs/sdk/api-reference/weave) class instance.

```ts
const instance = new Weave({
  ...
  plugins: [
    ...,
    new WeaveNodesSnappingPlugin(), // [!code ++]
  ]
})
```

</div>

<div className='[&_h3]:fd-step'>

### Use the plugin

Once the plugin is registered the final user now can snap the nodes when moved or resized
relative to other nodes or some guides like the center of the relative canvas.

</div>

</div>
