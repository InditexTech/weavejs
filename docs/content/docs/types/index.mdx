The `@inditextech/weavejs-types` package provides the shared TypeScript types and
constants used across the entire Weave.js ecosystem. It acts as a foundational
library that standardizes how data structures, node definitions, actions, plugins,
and stores are typed and referenced throughout your application and related packages.

Weave.js types package offers definitions for:
