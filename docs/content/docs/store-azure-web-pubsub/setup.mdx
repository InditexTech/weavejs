## Dependencies

### Backend-side dependencies

- [canvas](https://www.npmjs.com/package/canvas) package
- [@azure/web-pubsub](https://www.npmjs.com/package/@azure/web-pubsub) package
- [@azure/web-pubsub-express](https://www.npmjs.com/package/@azure/web-pubsub-express) package
- [y-protocols](https://www.npmjs.com/package/y-protocols) package
- [yjs](https://www.npmjs.com/package/yjs) package

### Frontend-side dependencies

This package is dependant on:

- [Weave.js types](/docs/types) package
- [Weave.js SDK](/docs/sdk) package

## Installation

```shell
npm install @inditextech/weave-store-azure-web-pubsub
```
